# YWifiControl

专为 **高通410 随身WiFi** 开发的 Windows 控制客户端，轻量、易用、开源，一键实现随身WiFi全功能管理。

## 功能亮点

- ✅ WiFi 快速扫描，实时展示可用网络
- ✅ 双击WiFi即可输入密码连接，操作便捷
- ✅ 实时监控设备状态（CPU占用、内存使用、IP地址、主机名）
- ✅ 深蓝专业UI界面，简洁美观，操作直观
- ✅ 单文件EXE打包，无需安装，直接运行
- ✅ MIT开源协议，免费使用、修改、分发

## 服务端（高通410 设备专用）

- 开发语言：PHP \+ Shell 脚本
- 核心优势：轻量无占用，不影响设备运行
- 管理方式：通过 NMCLI 工具实现WiFi全流程管理

## 客户端（Windows 专用）

- 开发语言：Python \+ Tkinter
- 界面风格：深蓝商务主题，视觉舒适
- 核心功能：支持自定义设备IP、通信KEY，灵活适配不同设备
- 操作便捷：双击WiFi列表即可发起连接，无需复杂指令

## 默认配置信息

- 设备默认IP：192\.168\.68\.1
- 通信密钥（KEY）：wifi123456
- 客户端设置密码：123456

## 部署教程（高通410 设备 \+ Windows 客户端）

1. 将项目 `server` 目录下的 3 个 PHP 文件（scan.php、connect.php、status.php），上传至高通410设备的 `/www/` 目录。
2. 将项目 `server` 目录下的 `wifi-scan.sh` 文件，上传至高通410设备的 `/usr/local/bin/`目录。
3. 在高通410设备上运行 Apache 服务，确保服务正常启动。
4. 打开 Windows 客户端（单文件EXE），默认配置即可连接设备，如需修改IP/KEY，输入设置密码（123456）后调整。
5. （可选）设置 Apache 开机自启，执行命令：`systemctl enable apache2`，重启设备后自动生效。
